CutBits
=======

Split hundreds of binary bits into integers by specific size.

Input/输入
----------

十六进制字符表示的二进制位，或160位，或224位。

Output/输出
-----------

或23位或24位一组的整数数组。

Usage/用法
----------

<code>
gcc -o cutbits main.c

./cutbits <HEX_STRING> ### <HEX_STRING>是二进制位的十六进制表示，如5A4B3C2D1E9F7D4C。
</code>
